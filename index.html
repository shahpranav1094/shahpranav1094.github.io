Harmonising Wi-Fi Direct Scanning Phase with Wi-Fi Scanning Phase

Summer Internship
Samsung Research and Development, Bangalore

Wi-Fi Direct connection is established when two devices go to a separate Wi-Fi direct mode and then probe requests and responses are exchanged. This process is generally slow and consumes additional power. The aim of this project was to harmonise the scanning phase of Wi-Fi Direct with that of Wi-Fi. Hence, the devices in question only needed to be connected to the same access point to establish a peer to peer connection between themselves.

Mentor: Vimal Edwin Bastin Joseph, Samsung Research and Development, Bangalore.

    Steganography

    Computer Science and Engineering, IIT Kharagpur

    Steganography ​is the practice of concealing a file, message, image, or video within another file, message, image, or video. In this project, any data was compressed using the zlib compression/decompression library. The compression ratio for huge data is very high which helps store a lot of information in an image without distorting the image. This compressed data was then encoded in the image following a particular pattern. An algorithm to retrieve the data hidden in the images was also created and then this data was decompressed using zlib libraries to obtain the original string encoded.

    Guide : Prof. Shamik Sural, Computer Science and Engineering, IIT Kharagpur

    Report: Steganography

    Text to Braille Converter

    Inter-Hall Hardware Exhibition

    Text to Braille Converter was a device which took A4 sheet printed pages over which a CMOS sensor would scroll over to get the image. The image was then converted to a character whose braille was then printed on a platform driven by 30 servo motors. All processing was performed on 8-bit ATMEGA microcontrollers.

    I designed the OCR module which used naive but fast method so that it can work on 8-bit microcontrollers, testing it initially using openCV and then shifting to AVR. Data transmission of images was also optimised by converting to Bayer for faster communication.

    Intensity Control Of Street Lights For Power Saving

    Schematic Diagram using Eagle

    PIR sensor is used to detect crossing of Vehicles. As soon as the vehicle crosses the sensor, the intensity of leds is increased for few seconds till the vehicle crosses the light. Coding is done in such a way that when there is no vehicle detection the output of sensor is low and the intensity of leds is 25% of its full intensity and when vehicle is detected the output of sensor goes high and then intensity of leds goes upto 90% of its original value.

    Report: Intensity Control Of Street Lights For Power Saving
